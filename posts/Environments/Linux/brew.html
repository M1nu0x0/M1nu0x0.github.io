brew 설치 가이드

Homebrew란?

맥스 호웰이 개발한 맥OS용 패키지 관리자.

Homebrew 설치

apt
sudo apt update && sudo apt upgrade -y

curl을 이용하여 설치를 할 것이니, 우선 curl을 다운 받도록 하자.

apt
sudo apt install curl -y && sudo apt install git -y

curl을 이용하여 아래 script를 다운받고 실행하자.

sudo 권한이 있다면 Homebrew의 의존성을 설치해달라고 한다.

apt-get
sudo apt-get install build-essential

Homebrew 확인

brew
brew --version
Homebrew 4.4.14

or

brew
brew doctor
Your system is ready to brew.

Homebrew 업데이트

brew
brew udpate
==> Updating Homebrew...
Already up-to-date.

Homebrew 삭제

Bash
curl -fsSL -o uninstall.sh https://raw.githubusercontent.com/Homebrew/install/master/uninstall.sh
Bash
bash uninstall.sh

이후 ~/.bashrc 하단 내용을 지우면 된다.

...

eval "$(/home/linuxbrew/.linuxbrew/bin/brew shellenv)"

brew Install test

- We recommend that you install GCC:
    brew install gcc

명령어 추천으로 GCC 설치를 해보라고 한다.

brew
brew install gcc
